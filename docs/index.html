Cómo diseñar un sitio web

Comenzar
→→

  1. Elige una de las plantillas de sitios web de Squarespace, que son las mejores de su clase.

  2. Usa nuestro creador de sitios web para personalizar las páginas, los menús y la navegación.

  3. Cambia las fuentes, las imágenes y los márgenes con el kit de ajustes de estilo integrado.

  4. Accede a una vista previa de tu diseño web adaptativo en los dispositivos móviles.

  5. Crea un logotipo para tu marca con nuestro creador de logos gratuito.

  6. Añade un dominio personalizado a tu sitio web.

Squarespace tiene todo lo necesario para crear una página web atractiva.
